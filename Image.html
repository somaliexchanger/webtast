Routes API

API Routes waxay hoy u tahay joogsi badan waxayna noqon kartaa shakhsiyaad gaariga aad doorbidayso. Taageerada wacyi-gelinta gaadiidka iyo ayna maraan wadooyinka, API-ga oo isdaba-jooga, API-ga ayaa u suurtogelinaya shakhsiyaadka iyo ganacsatada inay u safraan qaab ku habboon.

Abuurayaasha iyo shaqaalaha saadka waxay badbaadiyaan waqtiga iyo shidaalka iyagoo qorsheynaya wadooyin ay ku daraan xogta gaadiidka ee waqtiga dhabta ah. Si caddeymo wanaagsan, waxaad sidoo kale ku dahaadhi kartaa waddooyinkaaga in ka badan sawirro sarreysa oo ay suurtogal ka tahay xogta aan caadi ahayn.

Isochrone API

Horumarinta kabaalisaa Isochrone API si loo xisaabiyo meelaha safarka ee masaafo cayiman ama shuruudda waqtiga, bixinta waqtiga safarka ama polygon ku salaysan fog si ay u sawiraan qaabka aaggaas oo khariidad ah. Features sida taageero u hababka kala duwan ee safarka iyo xuduudahoodu customizable dhigi this API firfircoon.

API-ga hirgelinta tani waxay faa'iido u tahay dadka safarada ku jira safarrada ku lug leh noocyo badan oo gaadiidka ah iyo sidoo kale bixiyeyaasha saadka ee raadinaya qeexitaanka gobolka ee

Snap ilaa API Waddada

SNAP to Road API waa hab ku haboon oo lagu beddeli karo isku-dubaridka GPS ee waddooyinka dhabta ah iyadoo la adeegsanayo xogta xaddidan. Halkii ay gacanta ku shaqeeya iyada oo loo marayo noocyo badan oo iskudubbarid iyo iyaga qorsheynaya on map a, maamulayaasha raxan isticmaali kartaa API this si ay u toosin iyo dar qiimaha la muuqaalada dheeraad ah.

Taageerida habab badan oo gaadiidka ah iyo qorshaynta waddooyinka badan, SNAP to Road API waxay tixgelinaysaa xaddidaadda xawaaraha maxalliga ah si ay u aragto waddooyinka la xiriira degaanka qofka isticmaala.

API Wax-qabadka Multi-itinerary dhiciyaan

Maareeyayaasha raxanta ayaa si fudud u kormeeri kara baabuurta badan iyo xaalado isdabajoog ah oo adag oo leh Waxqabadka Multi-itinerary Optimization (MIO) oo ku dhacaya API.

MIO waa adeeg REST ah in iswada habraaca saadka muhiimka ah, caawinaya dadka isticmaala si fudud u dhiso itineraries for wakiilada dhowr ah halka ay tayadoodii ay wadooyinka ay. MiO waxay taageertaa kumanaan wakiil oo ku baahsan goobo badan oo waqti ah, waxayna leedahay codsiyo ku saabsan adeegyada qorshaynta iyo diridda labadaba.
